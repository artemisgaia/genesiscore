# Canva Product Page Copy Blocks

Use one product per Canva page.

## Required blocks

1. Hero
- Product name
- Functional category badge
- Suggested price
- One-line tagline

2. Clinical Summary
- What it is
- What it does
- How to use

3. Ingredient Focus
- Active ingredients (short list)
- Optional dosage line (only when verified)

4. Trust + Standards
- Third-party tested
- GMP-aligned manufacturing
- Clean excipient policy

5. Safety + Cautions
- Required cautions line
- Not medical advice line

6. Commerce CTA
- Add to cart
- Build my stack

## Brand voice guardrails

- calm, technical, executive
- no hype words
- no unverified medical claims
- keep lines short
- prefer explicit structure: what it is / what it does / how to use
